# Flask Registration System

This project is a simple Flask web application for user registration.

##  Features
- User registration form (Name, Email, Password)
- Form validation using Flask-WTF
- Error messages display
- Data stored using SQLite (SQLAlchemy)
- Confirmation page after submission

##  How to Run
1. Install dependencies  
   pip install flask flask-wtf flask-sqlalchemy  

2. Run the app  
   python app.py  

3. Open in browser  
   http://127.0.0.1:5000  

## Improvement
This is an improved version of my basic Flask registration project with added validation and database integration.

## Tech Used
Flask | Flask-WTF | SQLAlchemy | HTML
